# Forecast

A React application that can forecast whether to reorder new product stocks based on current inventory, average sales per week, and days to replenish the product. It uses Tensorflow.js to predict whether to reorder or not.

The source code of the Backend can be seen here: https://github.com/CSselrahC/appdev-final-act2-backend
